### Project Overview
Engineered a high-performance analytics architecture separating OLTP (parking transactions) from OLAP (analytics) workloads.

### Architecture & Tech Stack
* **Database Architecture:** Separated OLTP and OLAP workloads
* **Analytics:** Data warehousing for parking analytics
* **Performance Optimization:** Query optimization, indexing strategies
* **Reliability:** Eliminated system failures through architecture redesign

### Key Outcomes
* Eliminated system failures
* Improved query response times for hospital parking data
* Separated transactional and analytical workloads for better performance
