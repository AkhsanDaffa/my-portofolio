### Project Overview
Built an IoT solution enabling robot to follow black lines on white tiles.

### Architecture & Tech Stack
* **Controllers:** Raspberry Pi, ESP32
* **Computer Vision:** OpenCV for line detection
* **Motor Control:** PID-controlled motor navigation
* **Real-time Processing:** Low-latency image processing
* **Sensors:** Webcam

### Key Outcomes
* Achieved real-time line following processing
* Implemented PID control for smooth motor navigation
* Successfully navigated black lines on white tiles
